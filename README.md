# Requirements
    - Pip
    - Python3
    - Pygame

# How to install requirements
    - [Pip](https://pip.pypa.io/en/stable/reference/pip_install/)
    - [Python3](https://www.python.org/downloads/)
    - Pygame
        *  MacOs/UNIX `python3 -m pip install pygame`
        *  Windows `python pip install pygame`
